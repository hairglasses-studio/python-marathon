description: Start tutoring on the next unsolved exercise
---

1. Run `python marathon.py status` to find the next unsolved exercise.
2. Read that exercise's `README.md`, `problem.py`, and `test_problem.py`.
   **Never read `.meta/solution.py`** (denied by settings.json anyway).
3. Give the learner a brief orientation: the problem in one sentence, the
   target time, the signature they need to fill in, and one guiding
   question to get them started. Do not write any code yet.
