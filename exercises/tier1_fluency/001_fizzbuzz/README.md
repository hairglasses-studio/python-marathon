## 1.6 Exercise — Fizzbuzz (warm-up)

**Task:** Implement `fizzbuzz(n)` returning a list of strings for `1..n`:
- `"FizzBuzz"` if divisible by both 3 and 5
- `"Fizz"` if divisible by 3
- `"Buzz"` if divisible by 5
- the number (as a string) otherwise

**Why this exercise:** pure muscle memory — confirms you can write a `for` loop with conditions in your sleep.
